Introduction

Welcome to the Igor Database documentation. Here you will find information on how to use the API and other details about the database.

API Documentation

Here are the available API endpoints:

Creating a Collection


curl -X POST http://127.0.0.1:5000/create_collection \
    -H "Content-Type: application/json" \
    -d '{"collection_name": "my_collection"}'

Inserting a Document


curl -X POST http://127.0.0.1:5000/insert_document \
    -H "Content-Type: application/json" \
    -d '{"collection_name": "my_collection", "document": {"name": "igor", "age": 34}}'

Finding Documents


curl -X POST http://127.0.0.1:5000/find_documents \
    -H "Content-Type: application/json" \
    -d '{"collection_name": "my_collection", "query": {"name": "igor"}}'

Updating a Document


curl -X POST http://127.0.0.1:5000/update_document \
    -H "Content-Type: application/json" \
    -d '{"collection_name": "my_collection", "query": {"name": "igor"}, "update": {"age": 35}}'

Deleting a Document


curl -X POST http://127.0.0.1:5000/delete_document \
    -H "Content-Type: application/json" \
    -d '{"collection_name": "my_collection", "query": {"name": "igor"}}'

Deleting All Documents


curl -X POST http://127.0.0.1:5000/delete_all_documents \
    -H "Content-Type: application/json" \
    -d '{"collection_name": "my_collection"}'

Deleting a Collection


curl -X POST http://127.0.0.1:5000/delete_collection \
    -H "Content-Type: application/json" \
    -d '{"collection_name": "my_collection"}'
