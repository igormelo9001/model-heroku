About Igor Database

Igor is a simple NoSQL database designed to store JSON documents in text files. It aims to be lightweight and easy to use, making it ideal for small projects or learning purposes.

With Igor, you can create collections, insert documents, and query data using a simple API. All data is stored as JSON files, ensuring a straightforward and human-readable format.

Igor is an open-source project and contributions are welcome. Feel free to submit issues, suggest features, or contribute code on our GitHub repository.
