Clonar o Repositório Igor Database

Para obter o código-fonte do Igor Database, siga as instruções abaixo:

1. Obter o Repositório

# Clone o repositório utilizando Git
git clone https://github.com/seu_usuario/igor-db.git

2. Navegar para o Diretório do Projeto

# Navegue até o diretório clonado
cd igor-db

3. Configurar o Ambiente

# Crie e ative um ambiente virtual
python -m venv venv
source venv/bin/activate  # No Windows use: venv\Scripts\activate
# Instale as dependências pip install -r requirements.txt

4. Iniciar o Banco de Dados

# Execute o servidor Flask
python -m igor.api

Após seguir esses passos, você poderá iniciar e utilizar o Igor Database conforme descrito na documentação.
